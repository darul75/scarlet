Now lets take a look at how we use Scarlet to listen to this function once it is called. First we have to instantiate Scarlet. After that we have to tell Scarlet we would like to listen to Math.min. Once done we then tell Scarlet where to call to once this function is called.

var scarlet = new Scarlet(); Math.min = scarlet .intercept(Math.min, scarlet.FUNCTION) .using(function(info, method, args){ var result = method.call(this, info, method, args); print("Math.min called with args=" + args + " and result=" + result); return result; }).proxy(); var result = Math.min(1,2,3);
